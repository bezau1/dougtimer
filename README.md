# How to use
1. Clone repo
2. Go to OBS and go to Tools>Websocket Server Settings and turn on the websocket server
3. Copy your password from the websocket server menu and replace INSERTPASSHERE in obs_config.py
4. Create a scene called "/// Countdown Timer"
5. Create a group
